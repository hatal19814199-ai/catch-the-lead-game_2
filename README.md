# Поймай лид

Браузерная мини-игра для digital-проекта или сайта агентства: с неба падают **съедобные** предметы (апельсин, лимон, яблоко) и **несъедобные** (чайник, мяч, ракетка). Нужно ловить фрукты корзиной и не попадать под «не еду». Есть счёт, таймер, «репутация» (жизни) и экран окончания раунда.

## Стек

- HTML5, CSS3
- Canvas 2D + JavaScript (без сборщика и фреймворков)
- Шрифт [Outfit](https://fonts.google.com/specimen/Outfit) (Google Fonts)

## Структура репозитория

| Файл / папка | Назначение |
|--------------|------------|
| `index.html` | Разметка, HUD, легенда, старт и финиш |

## Запуск локально

### Вариант 1: открыть файл

Дважды откройте `index.html` в браузере. Шрифты подгружаются с Google — нужен интернет.

### Вариант 2: простой HTTP-сервер

Из папки проекта:

```bash
python -m http.server 8080
```

Затем в браузере: `http://localhost:8080`.

### Вариант 3: npm

Нужен [Node.js](https://nodejs.org/). Из корня проекта:

```bash
npm start
```

Команда через `npx` поднимет статический сервер (часто порт **3000** — смотрите вывод в терминале). Отдельный `npm install` не обязателен.
